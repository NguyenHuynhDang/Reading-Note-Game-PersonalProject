# Reading-Note-Game
A game that helps memorize and read musical pitch A B C D E F G

----How to play----

press the key responding the pitch apprearing on screen
S key for A pitch
D key for B pitch
F key for C pitch
Space for D pitch
J key for E pitch
K key for F pitch
L key for G pitch
